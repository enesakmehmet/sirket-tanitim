# 🏢 Modern Kurumsal Tanıtım Portalı

Bu proje, modern bir işletmenin dijital varlığını en profesyonel şekilde yansıtmak için Next.js 15 ve Tailwind CSS 4 kullanılarak geliştirilmiş tam kapsamlı bir kurumsal web sitesidir.

## ✨ Öne Çıkan Özellikler
- 🚀 **Next.js 15 Gücü:** App Router ve Turbopack ile ultra hızlı deneyim.
- 📱 **Tam Responsive:** Tüm cihazlarda (Mobil, Tablet, Masaüstü) kusursuz görünüm.
- 🎨 **Modern Tasarım:** Tailwind CSS 4 ile hazırlanmış, estetik ve profesyonel UI.
- 🌑 **Dark Mode:** Yerleşik gece/gündüz modu desteği.
- 📄 **Zengin Sayfa İçeriği:** Hizmetler, Projeler, Kariyer ve Blog gibi tüm kurumsal sayfalar.
- 🧱 **Hazır Bileşenler:** İstatistik sayaçları, müşteri yorumları carouselleri ve ekip bölümleri.

## 🛠️ Teknik Detaylar
- **Framework:** Next.js 15 (Latest)
- **Styling:** Tailwind CSS 4
- **Language:** TypeScript
- **Deployment:** Netlify / Vercel Ready
